# tic-tac-toe-ai
AI-Based Tic-Tac-Toe using Minimax Algorithm


## Description
This project implements an AI-based Tic-Tac-Toe game using the Minimax algorithm.
The AI evaluates all possible game states and makes optimal decisions, ensuring it never loses.

## Challenges Covered
- Implemented optimal decision-making using Minimax algorithm
- Created a Command Line Interface (CLI) for user interaction
- Tracked human wins, AI wins, and draw results

## Tech Stack
- Python
- Minimax Algorithm

## How to Run
1. Clone the repository
2. Run the following command:

python tic_tac_toe_ai.py
